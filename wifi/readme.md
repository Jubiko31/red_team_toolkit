## Wi-Fi Harvester

Gather all WiFi SSIDs:Passwords connected on target's machine. (Not a hacking tool).

Note: Oprimized for Windows. 

### Run
```sh
$ python wifi_harvest.py -o <OUTPUT_FILE>
```
